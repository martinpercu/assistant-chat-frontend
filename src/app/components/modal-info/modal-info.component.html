How this works
Our chat system features three unique AI agents, each with their own personality and area of expertise. You can select an agent from the menu on the left:
🤴 Napoleon's Funny Professor – Only answers questions related to the French leader.
📜 The Presidential Sage – Only answers questions related to the U.S. presidential history.
👩‍🏫 Professor Lisa – A knowledgeable teacher focused solely on Lisa.
🔍 A Shared Knowledge Base, But Specialized Agents
All agents use the same vector database built from PDF documents, but each is configured to respond only within their specific area—Napoleon, U.S. presidents, or Lisa. This keeps responses focused and relevant. You can access the original documents anytime via the "Docs" link.
💾 Independent Chat Memory
Each agent keeps its own chat history, remembering only what you share with them. For example, you could tell one agent your name is Peter and another that it's Mary, and each will remember it only in their own conversation.
